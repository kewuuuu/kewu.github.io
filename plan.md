# 博客项目计划

## 已实现功能

### kewu.github.io (博客网站)
- 文章列表树形结构展示（按分类和层级）
- 文章详情页面，包含：
  - 图片自适应屏幕，自动换行
  - 点击图片可大图查看并支持缩放/拖拽
  - 代码块展示（带头框/体框、行号、复制按钮、展开/收起功能）
  - 左侧目录（TOC）可展开收起，点击快速跳转
  - 面包屑导航显示文章路径
  - 响应式布局（横屏/竖屏适配）
  - 分栏可拖拽调整宽度，支持折叠

### kewu-cms (编辑器工具)
- 文章管理主界面：
  - 树形结构/扁平列表双模式切换
  - 支持按标题或更新时间排序
  - 支持正序/倒序
  - 点击文章可打开编辑器
- 文章编辑器界面：
  - 四栏布局：文章列表 | 目录 | 编辑区 | 预览
  - 所有分栏可拖拽调整宽度，支持折叠
  - 实时预览功能
  - 支持新建/编辑文章
  - 自动保存文章元数据（标题、描述、正文、分类、父级等）
- Tauri 桌面应用集成，支持本地文件系统操作

### kewu-posts (文章仓库)
- 文章采用独立 Git 仓库管理
- 支持文章层级结构（parent 字段）
- 支持分类管理
- 每篇文章独立文件夹，包含 index.md 和 assets
